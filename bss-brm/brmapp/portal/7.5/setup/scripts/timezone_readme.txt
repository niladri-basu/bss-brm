# entry from the list under TZNAME heading and use that in the timezone.cfg file's 
# TIMEZONE_VALUE Parameter.

# CAUTION:::Please make sure you use the Timezone value in timezone.cfg file.
# Collections script execution will fail without this setting done. 
# Its very necessary.
